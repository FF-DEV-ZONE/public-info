# 訊號輸出定義

- 文檔: [`全新UI (交易类）@ 监控-趋势开始25W46`](https://app.diagrams.net/#G1YUDk6MMF3ednKSfjuohWaN02oqTdr0C3#%7B%22pageId%22%3A%22NrDPr8AEW3ktn4Wz5YYc%22%7D)

- 數據輸出
  - table: t_index_info_data
  - type: AGG_SIGNAL
  - key: "OSCI" | "FAST_SLOW" | "UP_DOWN" | $\color{red}{"POS\\_NEG\\_CORREL"}$
  - symbol: "STOCK" | "COMMODITY" | "CN_COMMODITY" | "FX" | "F_AND_CN_BOND" | "FX_BOND"
  - content: 
  ``` javascript
  {
      signal: "TURN_UP" | "TURN_DN", "TREND_UP" | "TREND_DN" | "LONG_STRUCT" | "SHORT_STRUCT" | "LONG_EXPOSURE" | "SHORT_EXPOSURE" | 'NA'
  }
  ```
  
  example:
  ``` javascript
  {
    turn: NA,
    trend: NA,
    struct: NA,
    exposure: NA,
  }
  ```

- 目前支持的信号

  | 信号 | 产品 | 类型 |
  | --- | --- | --- |
  | OSCI | STOCK | turn |
  | OSCI | COMMODITY | turn |
  | OSCI | CN_COMMODITY 暫時不能用) | turn |
  | OSCI | F_AND_CN_BOND | turn |
  | FAST_SLOW | STOCK | turn, trend, struct |
  | FAST_SLOW | COMMODITY | turn, trend, struct |
  | FAST_SLOW | CN_COMMODITY （暫時不能用) | turn, trend, struct |
  | FAST_SLOW | F_AND_CN_BOND | turn, trend, struct |
  | UP_DOWN | STOCK | turn, trend, exposure |
  | UP_DOWN | COMMODITY | turn, trend, exposure |
  | UP_DOWN | CN_COMMODITY （暫時不能用) | turn, trend, exposure |
  | UP_DOWN | F_AND_CN_BOND | turn, trend, exposure |
  | $\color{red}{POS\\_NEG\\_CORREL}$ | $\color{red}{STOCK}$ | $\color{red}{turn, trend}$ |
