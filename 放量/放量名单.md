# 放量相关

## 放量定义
### 1 比例放量
- 说明: 单个小时在20天维度上的比例放量
- 数据查询
  ```sql
  SELECT toDateTime(x.time_close /1000, 'Asia/Shanghai'),toDateTime(x.create_time/1000, 'Asia/Shanghai'), x.symbol, x.signal_value
  FROM real_market.t_index_info_data x
  WHERE type = 'indicator_volume_rate'  
  and signal_key = 'vol_rate'
  ORDER BY x.create_time DESC
  ```
  
### 2 排名放量
- 说明: 20min的移动窗口的7天放量排名
- 数据查询
  ```sql
  SELECT toDateTime(x.time_close /1000, 'Asia/Shanghai'),toDateTime(x.create_time/1000, 'Asia/Shanghai'), x.symbol, x.signal_value
  FROM real_market.t_index_info_data x
  WHERE type = 'indicator_volume_section_agg'  
  and signal_key = 'section_volume'
  ORDER BY x.create_time DESC
  ```

### 3 独立放量
- 说明: 组别中的turnover roc(20min维度)突出的独立放量
